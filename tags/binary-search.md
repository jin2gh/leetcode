## 二分查找
### 算法思路
对于要查找的有序数组 nums 和目标数 target，通用步骤如下：
1. 定义左右指针 `l = 0, r = len(nums) - 1`；
2. 定义中间指针 `m = l + ((r - l) >> 1)`，使用减法避免最大值溢出；
3. 如果 `nums[m] == target`，则返回 `m`； 
4. 如果 `nums[m] > target`，则 `r = m - 1`；
5. 如果 `nums[m] < target`，则 `l = m + 1`；
6. 重复步骤 2-5，直到 `l > r`。
