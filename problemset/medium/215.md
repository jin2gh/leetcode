## Kth Largest Element in an Array / 数组中的第K个最大元素(https://leetcode-cn.com/problems/kth-largest-element-in-an-array/)

### 题解
#### 思路1：对数组从大到小排序后，给出第 k 个值
```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var findKthLargest = function(nums, k) {
  nums.sort((a, b) => (b - a))
  return nums[k - 1]
};
```
