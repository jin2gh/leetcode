## [最长快乐字符串](https://leetcode-cn.com/problems/longest-happy-string/)
### 题解
+ [TS](../../ts/1408/1405.ts)

#### 贪心
**思路**
题目要求最长的字符串，需要优先使用 `abc` 3 个字母中，数量最多的那个，添加字母时，需要判断当前要添加的字母是否与前两个字母相等。

**复杂度分析**
+ 时间复杂度：O((a + b + c) * C * logC)，`C = 3`，每次添加字母前需要按字母数量从大到小排序，时间复杂度为 O(C * logC)，最多排 `a + b + c` 次。
+ 空间复杂度：O(C)，`C = 3`。
