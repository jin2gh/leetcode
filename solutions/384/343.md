#### 数学
**思路**
对于整数 n，数学推导法可以得到，如果 n < 4，最大拆分的乘积为 `n - 1`；其余情况如下：
+ 如果 `n % 3 = 1`，最大乘积 `3^((n - 4) / 3)`；
+ 如果 `n % 3 = 2`，最大乘积 `3^((n - 2) / 3)`；
+ 如果 `n % 3 = 0`，最大乘积 `3^(n / 3)`。

**复杂度分析**
pow() 函数的时空复杂度与 CPU 支持的指令集相关。
