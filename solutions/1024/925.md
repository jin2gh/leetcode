#### 双指针
**思路**
+ 判断两个字符串的长度，如果 name 的长度小于 typed 的长度，直接返回 false。
+ 设置两个下标 i、j，判断 name 和 typed 对应的值是否相等，如果相等，i、j 自增；如果不等，判断 typed 当前字符是否等于前一个字符，相同则 j 自增，否则字符串不等。
+ 最后判断是否将 name 字符串遍历完成。

**复杂度分析**
+ 时间复杂度：O(n + m)，n、m 为两个字符串的长度。
+ 空间复杂度：O(1)。
